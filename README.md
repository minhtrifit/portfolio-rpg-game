## 📦 Cài đặt dự án

Cài đặt các gói & thư viện cần thiết

```console
npm install
```

hoặc cài đặt với legacy peer dependencies.

```console
npm install --legacy-peer-deps
```

Khởi chạy dự án (supported by [Vite](https://vitejs.dev/guide/))

```console
npm run dev
```
